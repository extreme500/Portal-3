**Função `ComputeWorldAABB()`** (linhas 2324-2347):
```cpp
CollisionAABB ComputeWorldAABB(glm::vec3 local_min, glm::vec3 local_max, 
                                const glm::mat4& model)
{
    // Transforma 8 cantos da bounding box local para coordenadas de mundo
    glm::vec3 corners[8] = { ... };
    glm::vec3 world_min = max();
    glm::vec3 world_max = -max();
    for (const glm::vec3& corner : corners) {
        glm::vec3 world_corner = glm::vec3(model * glm::vec4(corner, 1.0f));
        world_min = glm::min(world_min, world_corner);
        world_max = glm::max(world_max, world_corner);
    }
    return CollisionAABB{ world_min, world_max };
}
```

### **RECOMENDAÇÕES PARA API DE PORTAIS**

Para que a instanciação de portais seja **intuitiva e fluida**, sugiro:

1. **Criar um enum `PortalWall`** mapeando as paredes conhecidas:
   ```cpp
   enum PortalWall { 
       SALA1_FRONT, SALA1_BACK, SALA1_LEFT, SALA1_RIGHT,
       SALA2_LEFT, SALA2_RIGHT, SALA2_FRONT, SALA2_BACK,
       // ...
   };
   ```

2. **Função helper para calcular posição em parede:**
   ```cpp
   glm::vec3 GetPortalPositionOnWall(PortalWall wall, float uOffset, float vOffset);
   ```

3. **Struct Portal com matriz local-to-world:**
   ```cpp
   struct Portal {
       glm::vec3 position;
       glm::mat4 model_matrix;
       PortalWall attached_wall;
       glm::vec3 normal;  // normal da parede
   };
   ```

4. **Instanciação simples:**
   ```cpp
   Portal p = CreatePortal(SALA1_FRONT, 0.5f, 0.5f);  // posição normalizada [0,1]
   ```

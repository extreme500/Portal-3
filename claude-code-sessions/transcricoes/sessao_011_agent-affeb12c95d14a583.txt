### 2. CONSTRUÇÃO E USO DE MATRIZES View e Projection

**Arquivo de funções de matriz:** `/Users/tobiascadonamarion/Documents/UFRGS/CG/Portal-3/include/matrices.h`

#### 2.1 Matrix_Camera_View (linhas 224-253)
```cpp
glm::mat4 Matrix_Camera_View(glm::vec4 position_c, glm::vec4 view_vector, glm::vec4 up_vector)
```

Constrói a matriz de visualização usando:
- `position_c`: posição da câmera no espaço mundo
- `view_vector`: vetor de direção de visão (normalizado)
- `up_vector`: vetor "para cima" (sempre `(0, 1, 0, 0)`)

Retorna matriz View que transforma coordenadas mundo para coordenadas câmera.

**Uso em main.cpp linha 1017:**
```cpp
glm::mat4 view = Matrix_Camera_View(camera_position_c, camera_view_vector, camera_up_vector);
```

#### 2.2 Matrix_Perspective (linhas 269-321)
```cpp
glm::mat4 Matrix_Perspective(float field_of_view, float aspect, float n, float f)
```

Parâmetros em main.cpp (linhas 1021-1024):
- `field_of_view = 3.141592f / 3.0f` (60 graus em radianos)
- `aspect = g_ScreenRatio` (razão largura/altura)
- `nearplane = -0.1f` (plano próximo, negativo por convenção)
- `farplane = -20.0f` (plano distante, negativo)

**Características:**
- Usa matriz de projeção perspectiva com divisão por w
- Aplica -M*P para manter w > 0 (necessário para clipping no OpenGL)
- Importante: valores de near/far são negativos (câmera olha em -z)

void DrawVirtualObject(const char* object_name)
{
    glBindVertexArray(g_VirtualScene[object_name].vertex_array_object_id);
    
    // Set bounding box uniforms
    glm::vec3 bbox_min = g_VirtualScene[object_name].bbox_min;
    glm::vec3 bbox_max = g_VirtualScene[object_name].bbox_max;
    glUniform4f(g_bbox_min_uniform, bbox_min.x, bbox_min.y, bbox_min.z, 1.0f);
    glUniform4f(g_bbox_max_uniform, bbox_max.x, bbox_max.y, bbox_max.z, 1.0f);
    
    // Draw call
    glDrawElements(
        g_VirtualScene[object_name].rendering_mode,
        g_VirtualScene[object_name].num_indices,
        GL_UNSIGNED_INT,
        (void*)(g_VirtualScene[object_name].first_index * sizeof(GLuint))
    );
    
    glBindVertexArray(0);
}
